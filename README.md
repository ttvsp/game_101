# Игра 101

Веб-приложение для игры в карточную игру "101" в Telegram Mini App.

## Особенности
- Темный современный дизайн
- Адаптивная верстка для мобильных и десктоп
- График динамики счета
- Интеграция с Telegram WebApp API

## Технологии
- HTML5
- CSS3
- JavaScript
- Chart.js
- Telegram WebApp API
